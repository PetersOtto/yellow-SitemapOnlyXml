# yellow-SitemapOnlyXml
SitemapOnlyXml generates a »sitemap.xml« for the Yellow CMS by datenstrom.

<p align="center"><img src="01-yellow-sitemaponlyxml.png?raw=true" alt="Screenshot yellow-SitemapOnlyXml"></p>

## Why?
There is already a very good [sitemap extension](https://github.com/annaesvensson/yellow-sitemap/tree/main) for the yellow CMS. However I only needed a `sitemap.xml`, without the `sitemap.html`. So I modified the [original extension](https://github.com/annaesvensson/yellow-sitemap/tree/main) slightly for this purpose, so that only the `sitemap.xml` is generated. 

## How to install an extension
Download ZIP file and copy it into your `system/extensions` folder. [Learn more about extensions](https://github.com/annaesvensson/yellow-update)
